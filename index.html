Quick start
  1. Ensure that no sequence is currently running.
  2. Reload the front panel webpage (http://localhost).
  3. Attach the manifold to the inlet system and plug in the corresponding yellow tubes.
  4. Pump the vented volume between the green valve on the manifold and V22. First use the scroll pump then the turbo.
  5. Once the vacuum reaches 0.0001 mbar, open the green valve manually.
  6. Click the violet Starting position button.
  7. To choose the measurement sequence, click the Sequence: Choose File button.
    • The sequence .csv files are usually in the /home/pi/Desktop folder.
    • You can modify the existing sampleSequence.csv file, or create a new file. The file name does not matter.
    • Modifying a sequence file after it is loaded in will not automatically update on the front panel. You have to refresh the front panel and upload the modified sequence.
  8. To start the measurement, click the green Start sequence button.

User manual

General notes

  • Before stopping the serialComm.py or the Arduino, make sure the Z bellow is at 50%!
  • To reprocess a single measurement: http://192.168.1.242/evaluateData.php?sampleName=221223_025949_test_Air&polynomial=100
  • To reprocess a multiple measurements: /usr/bin/python /var/www/html/Python/batchReprocess.py

Startup

  • To turn the TILDAS on: (1) plug it in, (2) toggle the power switch in the back, (3) turn on the main swith on the front.
  • The rPi has to be turned on after the TILDAS, otherwise reboot it: sudo shutdown -r now.
  • To allow streaming data from the TILDAS to the rPi, click the "RS" on (TDLWintel bottom left) and set the mode to "Stream Data" (TDLWintel top left).
  • Make sure that "RS" is active at TILDAS and that mode is in "Stream Data", otherwise Python does not get data.
  • Start the communication script: /usr/bin/python /var/www/html/Python/serialComm.py

Methods and sequences

The general structure of a command-line in the method files is "command,parameter,wait". You can edit the method .csv files using a text editor or VSCode. The parameter is used for most command types, ignored for some, but it always has to be present.
Sequences and methods should always start from a specific starting situation. You can reset the valves to this situation by pressing the Starting Position button.

  • V03,0,15, <--- Close valve 3 and wait 15 s
  • V13,1,12, <--- Open valve 13 and wait 12 s
  • WC,0,5, <--- Write current cell presure on front panel.
  • WA,0,7, <--- Write current pressure at gauge A on the front panel.
  • WR,0,12, <--- Write current reference CO2 pressure (gauge X) on the front panel. This value is reset after the method is finished.
  • WS,0,6, <--- Write current sample CO2 pressure (gauge Y) on the front panel. This value is reset after the method is finished.
  • CM,0,1, <--- Write current pCO2 on the front panel for mixing ratio adjustment. Parameter 0 is for reference gas and 1 for sample gas. This value is reset after the method is finished.
  • IA,363,7, <--- Introduces Air. Opens V15, waits until the pressure on gauge A reaches the value set in the parameter, then closes valves V15.
  • TWD,1,1, <--- Starts the TILDAS writing data.
  • TWD,0,1, <--- Stops the TILDAS writing data.
  • BX,45.3,12, <--- Move bellows X to 45.3%.
  • BX,-7.2,12, <--- Move bellows X by -7.2% relative to the current position.
  • BY,25.3,12, <--- Move bellows Y to 25.3%.
  • BY,+4.5,12, <--- Move bellows Y by 4.5% relative to the current position.
  • BZ,25.3,12, <--- Move bellows Z to 25.3%. If the parameter has a sign, the bellows are moved by an increment relative to the current position.
  • PX,1.653,10, <--- Move bellows X or expand the gas so that the pressure is 1.653 mbar in bellows X. The starting situation should be V05C, V07C, V06O.
  • PY,1.653,10, <--- Move bellows Y so that the pressure is 1.653 mbar in bellows Y. The starting situation should be V11C, V13C, V12O.
  • QC,39.387,10, <--- Set the cell pressure by moving bellows Z to the pressure written by command WC on the front panel. The parameter 39.387 is only used in case that there is no value given on the frontpanel.
  • SN,366,10, <--- Set the pressure at gauge A by moving bellows Z. The parameter (e.g., 366) is used, unless it is 0. Otherwise the value written by command WA on the front panel is used.
